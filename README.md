# WhatsApp API Manager Node.js

Aplikasi WhatsApp API Manager Node.js adalah aplikasi berbasis node js untuk kirim whatsapp secara ootomatis dengan menggunakan Api.
Aplikasi ini sudah dilengkapi fitur login dan dashboard untuk mengelola API dan disertai penyimpanan ke database.
